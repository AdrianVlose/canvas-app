# Canvas Application

## Extensions & Dependencies

For develop and run this project, you need to have:

- Node.js
- Live Server (VS Code extension)
- Live Sass Compiler (Vs Code extension for compiling scss files)

## Setup & Run

- create a folder with name *dist* in the root directory to store compiles JavaScript files from Typescript compiler
- _npm install_ (run this command only first time) 
- _tsc --watch_ (for running the Typescript compiler in watch mode )
- Open _index.html_ using **Live Server**
